# MCP Doris 工具

这是一个用于操作Apache Doris的MCP工具，提供了一系列用于管理和操作Doris集群的功能，同时支持通过LLM/MCP进行交互。

## 功能

- 连接到Doris集群
- 执行SQL查询
- 管理数据库和表
- 监控集群状态
- 导入和导出数据
- 支持LLM通过MCP协议调用

## 安装

### 全局安装

```bash
npm install -g mcp-tools-doris
```

### 通过npx直接使用

```bash
npx mcp-tools-doris <命令>
```

### 本地安装

```bash
npm install mcp-tools-doris
```
